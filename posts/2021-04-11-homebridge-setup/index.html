HomeBridge setup and supported accessories.

Homebridge instalation.

Simple way to install it, is using docker and docker compose.

 1version: '2'
 2services:
 3  homebridge:
 4    image: oznu/homebridge:ubuntu
 5    restart: always
 6    network_mode: host
 7    environment:
 8      - PGID=1000
 9      - PUID=1000
10      - HOMEBRIDGE_CONFIG_UI=1
11      - HOMEBRIDGE_CONFIG_UI_PORT=8581
12      - TZ=Canberra/Australia
13    volumes:
14      - ./volumes/homebridge:/homebridge

For more ways to install and for configuration options head to the official docs of HomeBridge here.

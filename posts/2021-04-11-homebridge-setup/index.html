April 11, 2021 · 1 min · 59 words · Me

HomeBridge setup and supported accessories.#

Homebridge instalation.#

Simple way to install it, is using docker and docker compose.

version: '2'
services:
  homebridge:
    image: oznu/homebridge:ubuntu
    restart: always
    network_mode: host
    environment:
      - PGID=1000
      - PUID=1000
      - HOMEBRIDGE_CONFIG_UI=1
      - HOMEBRIDGE_CONFIG_UI_PORT=8581
      - TZ=Canberra/Australia
    volumes:
      - ./volumes/homebridge:/homebridge

For more ways to install and for configuration options head to the official docs of HomeBridge here.
